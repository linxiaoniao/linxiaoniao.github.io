在mac上使用github+hexo搭建个人博客

发表于2021-01-04|经验分享
|字数总计:929|阅读时长:3分钟

该步骤在mac系统下搭建静态博客，可能与window下有所出入，仅供参考。

Hexo是什么

hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Coding上，是搭建博客的首选框架。


GitHub Pages是什么

GitHub pages 是GitHub免费提供给开发者托管个人网站代码的产品，得益于其特性，我们可以将个人网站托管在上面无需另外购买服务器，且可以绑定个人域名。

自定义域名

以下步骤以阿里云域名服务为例

  • 购买域名

购买域名途径很多，可以通过阿里云、腾讯云、GoDaddy等进行购买，正常来说搭建博客的话几块钱一年就可以购买到一个心水的域名。

阿里云域名链接

  • 解析至GitHub仓库

进入阿里云域名解析管理，添加一条CNAME记录，指向博客仓库ip地址（通过ping查询ip地址）

  • 在blog的source目录下新建CNAME文件

CNAME文件内填入自定义域名

重新hexo d推送到GitHub仓库生效后就可以通过域名访问到博客啦
